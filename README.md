# repo-therapist

A structured practice for diagnosing the health of a software codebase.

---

**Repo therapy** is a reflective diagnostic session — not a linter, not a
code review, not a refactoring plan. It produces a prioritized, honest
account of what is genuinely unhealthy in a codebase and what should be
addressed first.
